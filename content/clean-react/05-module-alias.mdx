# Configuração

Inclusão do parâmetro `paths` <mark>abaixo de **baseUrl**</mark> no `tsconfig.json`.

```json
// tsconfig.json

"paths": {
  "@/*": ["*"]
},
```

Inclusão do parâmetro `moduleNameMapper` no arquivo `jest.config.js`

```js
// jest.config.js

moduleNameMapper: {
  '@/(.*)': '<rootDir>/src/$1'
}
```
