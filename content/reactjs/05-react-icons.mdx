## Instalação

```bash
$ yarn add react-icons
```

## Utilização

**Importação do pacote (exemplo utilizando o Font Awesome)**

```js
import { FaGithubAlt, FaPlus } from 'react-icons/fa';
```

**Chamada do ícone**

```js
<FaPlus color="#FFF" size={14} />
```
